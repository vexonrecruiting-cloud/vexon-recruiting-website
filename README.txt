PUBLISH WITH GITHUB PAGES
1. Create a new public GitHub repository named vexon-recruiting-website.
2. Upload every file and the assets folder.
3. Open repository Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select main and /(root), then Save.
6. GitHub will provide a free website address.

CUSTOM DOMAIN
After the free site is working, add vexonrecruiting.ca under Settings > Pages > Custom domain.
Your domain provider will then need DNS records. Do not change DNS until the GitHub site is live.
